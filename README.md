# Does This Feel Right?

A minimalist blog with automatic email notifications.

## Features
- ✅ Static Site Generator (Python)
- ✅ User Accounts (Supabase)
- ✅ Save Posts & Notes
- ✅ Reading Progress Tracking
- ✅ Automatic Email Notifications
- ✅ Post Filtering by Category

## Deployment
- Hosted on Netlify
- Auto-deploys from GitHub
- Custom domain: www.doesthisfeelright.com

## How Email Notifications Work
When you publish a new post and push to GitHub, all users automatically receive an email with:
- Post title and excerpt
- Direct link to read
- Beautiful minimal design
